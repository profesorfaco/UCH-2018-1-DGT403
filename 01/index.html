JavaScript

Entre los lenguajes de programación para web, podemos diferenciar a unos que se ejecutan del lado del servidor, y otros que se ejecutan del lado del cliente.

Por tratarse de una introducción, nos limitaremos a explorar JavaScript como lenguaje de programación del lado del cliente. Esto implica que la ejecución de lo que programemos será responsabilidad de cada navegador.

1. Jugar con la consola de JavaScript del navegador

En cada navegador, la consola de JavaScript posee un aspecto determinado y se abre de forma diferente. Para abrirla, puedes utilizar el "shortcut" de teclado que corresponda:

  • Si estás usando Chrome en PC, presiona Ctrl + Shift + J. Si estás usando el mismo navegador en Mac, presiona Cmd + Alt + J
  • Si estás usando Firefox en PC, presiona Ctrl + Shift + K. Si estás usando el mismo navegador en Mac, presiona: Cmd + Alt + K
  • Si estás usando Safari, presiona: Cmd + Alt + C
  • Si estás usando otro navegador, no sabría como ayudarte con el "shortcut".

Cuando se muestre la consola, podrás leer un mensaje que resulta de concatenar lo que guardaba una variable con una cadena de caracteres entre comillas; me explico:

console.log(saludo + " Yo soy la consola de JavaScript");

Usando console.log() programas la impresión de un mensaje en la consola (este método puede resultar prescindible cuando trabajamos directamente sobre la consola, pero es mejor acostumbrarse a usarlo). El contenido del mensaje impreso será lo que incluyas dentro del paréntesis. En este caso, lo primero entre paréntesis es saludo, sin comillas. Eso es una variable, a la que se le encargó guardar la cadena de caracteres Hola mundo!. Lo que sigue es un signo + que concatenará esa cadena de caracteres con otra que se agrega, entre comillas, dentro del mismo paréntesis: " Yo soy la consola de JavaScript".

En el párrafo anterior y en este, la palabra variable tiene su primera sílaba destacada. Esto es para que sepas cómo se declara una variable en JavaScript: Anteponiendo un var a una palabra que será el nombre de la variable.

Importante es recordar que cuando le quieras encargar a la variable guardar una cadena de caracteres, debes usar comillas:

var sujeto = "Yo";
var adjetivo = "Lorem ipsum";
console.log(sujeto +" soy muy "+ adjetivo);

Pero cuando le quieras encargar números, no agregues comillas:

var primervalor = 20;
var segundovalor = 30;
console.log(primervalor + segundovalor);

¡Antes de que presiones Enter! Una pregunta: ¿Qué crees que resulte, 50 o 2030?

En caso que no le hayas acertado a la respuesta, podrías tener más suerte con lo que sigue: Piensa un número entre 1 y 10 (incluyendo ambos extremos).

console.log(Math.round((Math.random() * 10) + 1));

Eso pudo ser complicado, pero vamos por partes, por paréntesis, de adentro hacia afuera:

  • La función Math.random() retorna un número no entero, pseudo-aleatorio, desde el 0 hasta casi el 1 (por eso hay que multiplicar por 10 y sumar 1).
  • La funcióin Math.round() retorna el número redondeado.

Tus opciones partían en 1, incrementándose tantas veces como lo permitiese la condición de menor o igual a 10:

for (var x=1; x<=10; x++){ 
console.log(x); 
}

Y desde aquí puedes empezar a complejizar o complicarte:

for (var x=1; x<=10; x++){
console.log(x); 
if (x == 3){console.log("I like turtles!");}
}

A las variables se les puede encargar guardar más que cadenas de caracteres y números. Eso lo podrías revisar en otra instancia. Por ahora, favor considera suficiente un ejemplo para probar en consola:

var bolso = ["computador", "lápices", "libreta", "alcohol gel"];
var compu = {modelo:"MacBook Pro", memoria:"10GB", os:"macOS Sierra"};
console.log(bolso[0] + " " + compu.modelo);

2. Desplegar una alerta con un click

Si presionas el botón, el navegador desplegará una alerta. Para hacerlo, basta con tomar el elemento HTML <button>…</button> y agregarle un atributo OnClick, que en su variable contenga la programación de la impresión de una alerta, cuyo mensaje se define entre paréntesis. En este caso, como en el primer mensaje que vimos en la consola, tenemos el resultado de concatenar lo que guardaba una variable con una cadena de caracteres entre comillas. Esto es:

<button onclick="alert(saludo + ' Yo soy una alerta de tu navegador')">el botón</button>
